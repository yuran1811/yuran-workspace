## Quick Start

**Prerequisites**

Make sure you have the following installed or downloaded on your machine:

- [Git](https://git-scm.com/)

**Cloning the Repository**

```bash
git clone --recurse-submodules -j8 https://github.com/yuran1811/yuran-workspace.git
cd yuran-workspace
```
